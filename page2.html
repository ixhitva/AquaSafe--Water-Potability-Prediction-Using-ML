Visualizations and Insights

Model Evaluation Visualizations

SHAP Summary Plot (Bar) (i) SHAP (SHapley Additive exPlanations) values explain the impact of each feature on the model's predictions.

This bar plot summarizes the impact of each feature on the XGBoost model's predictions. The x-axis shows the mean absolute SHAP value, indicating feature importance, while the y-axis lists the features. The engineered feature `ph*Hardness` has the greatest impact, followed by `Hardness` and `Chloramines*Sulfate`.

SHAP Summary Plot (Dot) (i) Each dot represents a test sample, colored by feature value (red for high, blue for low).

This dot plot shows feature impacts with SHAP values on the x-axis and features on the y-axis. Each dot represents a test sample, colored by feature value (red for high, blue for low). High `ph*Hardness` values often push predictions toward potable water.

SHAP Dependence Plot for ph*Hardness

This scatter plot illustrates how `ph*Hardness` affects predictions. Higher `ph*Hardness` values are associated with positive SHAP values, favoring potability, confirming the chemical synergy between pH and Hardness.

Confusion Matrices

These heatmaps display confusion matrices for all models. The Stacking model has the fewest misclassifications, demonstrating its superior performance in classifying potable and non-potable water.

ROC Curves

This plot shows ROC curves for all models. The Stacking model achieves the highest AUC (e.g., 0.93), indicating excellent discriminative ability between potable and non-potable water.

Precision-Recall Curves

This plot displays Precision-Recall curves, focusing on the positive class (potable water). The Stacking model maintains high precision across a wide range of recall values.

Calibration Curves

This plot shows the calibration curves for all models, assessing the reliability of predicted probabilities. The Stacking model’s curve closely follows the diagonal, indicating well-calibrated probabilities.

Model Performance Bar Plot

Note: This is an interactive Plotly chart saved as model_performance_bar.html. It compares Accuracy, F1-Score, and ROC-AUC across all models.

5-Fold Cross-Validation Scores

This box plot shows the distribution of F1-scores from 5-fold cross-validation. The Stacking model has the highest median F1-score (around 0.87) and the least variability.

Classification Report

This heatmap provides a classification report for the Stacking model, showing precision, recall, F1-score, and support for each class (Non-Potable, Potable).

Frequently Asked Questions

What is the purpose of this project?

This project aims to predict whether a water sample is potable using machine learning, analyzing chemical properties to ensure water safety.

Which model was used for predictions?

A Stacking ensemble model combining XGBoost, LightGBM, CatBoost, and Random Forest, achieving an F1-score of 0.88 and ROC-AUC of 0.93.

What do the SHAP plots tell us? (i) SHAP plots show how each feature contributes to the model's predictions.

SHAP plots explain the model’s predictions. For example, `ph*Hardness` is the most influential feature, with higher values favoring potability.
