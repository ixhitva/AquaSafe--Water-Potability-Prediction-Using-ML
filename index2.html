What is the purpose of this project?

This project aims to predict whether a water sample is potable using machine learning, analyzing chemical properties to ensure water safety.

Which model was used for predictions?

A Stacking ensemble model combining XGBoost, LightGBM, CatBoost, and Random Forest, achieving an F1-score of 0.88 and ROC-AUC of 0.93.

What do the feature importance plots tell us? (i) Feature importance plots show how each feature contributes to the model's predictions.
